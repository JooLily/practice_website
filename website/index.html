Links

The anchor element, "a", used with the hypertext reference attribute, "href", can be used to link a webpage or other files.

However, without a "target" attribute, the user who clicks on the anchored text will be redirected to the link on the same tab. Designating the "target" as "_blank" will open the link in a new tab.

IMPORTANT: Using the "rel" attribute to include "noopener" and "noreferrer" will ensure privacy and security of both the webpage and the user who accesses the link.

noopener
This attribute makes sure that the link opened in a new tab cannot access the page the user is coming from. Without this attribute, the new page may manipulate the new page with Javascript (this can pose a security risk).source
noreferrer
This attribute hides the referrer, preventing the new page from knowing where the user came from, and also includes the behavior of noopener, preventing the new page from accessing the original page.source
